# docs/transcripts

This directory holds 3-Amigos session notes and meeting transcripts related to product, development, and QA alignment.

## Purpose

Captures shared understanding from collaborative refinement sessions. Used as source material for generating or refining Jira tickets.

## Naming Convention

Use kebab-case with a date prefix: `YYYY-MM-DD-session-topic.md`

e.g. `2024-06-12-checkout-flow-3-amigos.md`

## Required Header Format

Every transcript file **must** begin with the following header:

```markdown
# [Session Title]

**Date:** YYYY-MM-DD  
**Attendees:** [Name (Role), Name (Role), ...]  

## Decisions

- [Decision made during the session]

## Open Questions

- [Unresolved item or question to follow up on]
```
